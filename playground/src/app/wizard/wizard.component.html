-- 👨🏻‍💻 Welcome to the noREST playground 👩🏻‍💻 --

This notonly.rest webpage is the playground to try out the reference implementation of noREST. More information about the noREST project can be found on its Github page.

You have 2 possible starting points:

  1. Creating an playground with pre-stored e-commerce sample data is the best way to explorer the functionalities of noREST
  2. An empty playground has no data stored yet and you can explorer noREST from the ground.
