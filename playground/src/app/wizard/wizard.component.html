Welcome to the noREST playground

This notonly.rest page is the playground to try out the reference implementation of noREST, a no-code (not only) RESTfull API. More information about the noREST project can be found on its Github page.

You have 2 possible starting points:

  1. Creating an playground with pre-stored e-commerce sample data is the best way to explorer the functionalities of noREST
  2. An empty playground has no data stored yet and you can explorer noREST from the ground.

If you want to return to an existing playground, simply attach your nr- key to the url:

You can access the API at any-time with your API-KEY @ https://api.notonly.rest/nr-xxxxxx. Note that playgrounds might get deleted if a certain data volume is reached.
